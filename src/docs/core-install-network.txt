wp-core-install-network(1) -- Transform a single-site install into a
multi-site install.
====

## SYNOPSIS

`wp core install-network` --title=<network-title> [--base_path=<url-path>]

## OPTIONS

* `--title`=<site-title>:

	The title of the new network.

* `--base_path`=<url-path>:

	Base path after the domain name that each site url will start with.
Default: '/'
